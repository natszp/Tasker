
#  Tasker 
A very simple task manager in Django. 

## Installation (on Ubuntu)
  
- django > 2.2.9,
- django-cors-headers > 3.2.1,
- Faker > 4.0.0,
- psycopg2 > 2.7.4,
- pgAdmin4

## Usage
Add, update and delete your daily tasks!
